## ğŸƒâ€â™‚ï¸ **Quick Start Guide**

### **Prerequisites**
- Node.js 18+ 
- npm or yarn package manager
- OpenAI API key (optional for AI features)

### **1. Clone & Install**
```bash
git clone https://github.com/JAGAN666/health-triage-app.git
cd health-triage-app
npm install
```

### **2. Environment Setup**
```bash
cp .env.local.example .env.local
```

Add your environment variables:
```env
# OpenAI API (optional - demo mode works without it)
OPENAI_API_KEY=your_openai_api_key_here

# Database (optional - SQLite used by default)
DATABASE_URL="file:./dev.db"

# Authentication (optional - demo mode available)
NEXTAUTH_SECRET=your_nextauth_secret_here
NEXTAUTH_URL=http://localhost:3000
```

### **3. Database Setup**
```bash
npx prisma generate
npx prisma db push
npx tsx prisma/seed.ts
```

### **4. Development Server**
```bash
npm run dev
```

Visit `http://localhost:3000` to see the application in action!

### **5. Production Build**
```bash
npm run build
npm start
```
